## 项目简介

**Open Arena Studio** 是一款免费的桌面应用，让你能够同时向多个 LLM（大语言模型）提供商发起编程挑战，并将对比结果导出为可分享的 PK 对战视频。

你只需输入一道编程题，应用会让你选择的各家 AI 模型同台竞技，实时生成代码，并自动录制整个过程，最终输出一段并排对比的精彩视频。

无论你是在评估哪款模型最适合你的开发工作流，还是想制作 AI 测评内容、或者只是好奇 GPT-4o 与 Claude、Gemini 在实际编程任务上的差距，Open Arena Studio 都能让这种对比变得直观、生动且易于分享。

---

## ✨ 核心功能

- **多模型 PK 对战** — 同时让两个或多个 LLM 提供商在同一道编程题上竞技。
- **自带密钥（BYOK）** — 接入你自己的 API Key，支持 OpenAI、Anthropic、Google Gemini、本地模型（Ollama、LM Studio）等。所有数据均在本地处理，不经过第三方服务器。
- **实时编程竞技场** — 并排实时观看每个模型流式输出代码的全过程。
- **一键导出视频** — 自动录制对战过程，生成随时可分享的 MP4 对战视频。
- **完全免费** — 下载即用，无需付费。
- **全平台支持** — 兼容 macOS、Windows、Linux。

## 快速开始

1. **下载并安装** — 从 [Releases 页面](https://github.com/kaying-studio/open-arena-studio/releases) 下载适合你系统的安装包并完成安装。
2. **添加 LLM 提供商** — 进入设置，填写你想对比的各家提供商的 API Key（OpenAI、Anthropic、Gemini、Ollama 等）。
3. **发起新对战** — 选择两个或多个提供商，输入一道编程题，点击 **开始**。
4. **观战竞技场** — 每个模型实时编码，应用自动录制全过程。
5. **导出视频** — 点击 **导出视频**，保存或分享生成的 MP4 对战视频。

## 支持的 LLM 提供商

| 提供商 | 说明 |
|---|---|
| OpenAI（GPT-4o、GPT-4.1、o3…） | 需要 OpenAI API Key |
| Anthropic Claude | 需要 Anthropic API Key |
| Google Gemini | 需要 Google AI API Key |
| Ollama / LM Studio | 本地模型，无需 Key |
| 任何 OpenAI 兼容接口 | 支持自定义 Base URL |
